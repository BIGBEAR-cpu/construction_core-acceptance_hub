# 审核报告模板（Review Report Template）

## 总体结论：PASS / CONDITIONAL PASS / FAIL
- 在此给出审核结论。

## 是否完成任务书目标
- 对照任务目标逐条说明完成情况。

## 是否越界修改
- 检查是否超出允许修改范围。

## 是否破坏已有功能
- 说明回归验证结果与潜在破坏点。

## 必须修复问题
- 列出阻断合并的问题。

## 建议修复问题
- 列出推荐优化但不阻断合并的问题。

## 可暂缓问题
- 列出可在后续迭代处理的问题。

## 测试覆盖评价
- 评价测试完整性、覆盖深度与关键缺口。

## 风险点
- 列出风险等级、影响与缓解建议。

## 是否建议合并
- 明确给出“建议合并 / 不建议合并 / 有条件建议合并”。
